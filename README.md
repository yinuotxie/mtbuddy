# MTBudy

MTClaw-inspired personal AI workstation for MTT AIBOOK.

MTBuddy is a local-first desktop AI workstation that turns natural language
requests into auditable work products: documents, spreadsheets, slide decks,
research notes, file operations, and knowledge-base answers. The project is
designed to run locally first, then plug into MTT AIBOOK, MTT AIOS, and MTClaw
when that environment is available.
