# MTBUDDY Competition Demo

## Why MTBUDDY
- Local-first personal AI workstation for MTT AIBOOK.
- OpenAI-compatible agents can call MTBUDDY through MTClaw Function Router.
- Every run produces inspectable artifacts and an audit log.

## Capability Packs
- Tools expose precise functions such as file, sheet, report, and PPTX actions.
- Skills describe when to use tools and how to verify the result.
- Workflow recipes stay optional so the user remains in control.

## Demo Flow
- Pick a workspace.
- Run the agent through MTClaw-backed MTBUDDY tools.
- Inspect report, action CSV, slide deck, and audit evidence.
